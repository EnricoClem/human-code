        - andare su amazon.it
            - SE hai un account
                - accedi
            - ALTRIMENTI

    - ALTRIMENTI
        - sbloccare il telefono

            - ALTRIMENTI

                    - SE hai un account

    - cercare lampada
        - controllo requisiti

            - AND nel budget
                - aggiungere al carrello
                - interrompi ricerca

                - proseguire con la lista
    - cercare penna
        - controllo requisiti

        - controllo requisiti

                - aggiungere al carrello

            - ALTRIMENTI
                - proseguire con la lista

- completare l'ordine
